# Python

根据定义我们可以写出：

```python
def f(n):
    if n == 0:
        return 0
    elif n == 1:
        return 1
    else:
        return f(n-1) + f(n-2)


if __name__ == "__main__":
    a = f(0)
    b = f(1)
    c = f(10)
    print(a)
    print(b)
    print(c)
```

优化过的斐波那契数列:

```python
k = {
    0: 0,
    1: 1
}

def f(n):
    if n in k.keys():
        return k[n]
    else:
        k[n] = f(n-1) + f(n-2)
        return k[n]


if __name__ == "__main__":
    import time

    start = time.time()
    for i in range(99):
        a = f(i)
        print(a)
    print(time.time() - start)
```

使用字典缓存之前已经计算过的斐波那契数，计算速度大幅提升。

# JavaScript

```javascript
function f(n) {
    if (n === 0) {
        return 0
    } else if (n === 1) {
        return 1
    } else {
        return f(n - 1) + f(n - 2)
    }
}

console.log(f(10))
```
